更“优雅”地访问Github

本文最后更新于：2021年4月24日 凌晨

前言

Github 是开源项目的圣地，但其在国内访问速度很慢，最主要的原因就是GitHub的分发加速网络域名遭到了DNS的污染。为了解决这个问题，可以通过修改Hosts文件，绕过国内的DNS解析，直接访问GitHub的CDN节点，从而达到加速的目的。

SwitchHosts

SwitchHosts是一款Hosts快速切换小工具，作用是用来一键切换Hosts配置文件，非常实用,操作起来非常简单。目前发布的版本为 v4.0 版本.它有着一些很不错的特性，比如：

  • 免费、开源、支持三大操作系统
  • 系统托盘快速「一键切换」不同的 hosts 方案
  • 支持 hosts 文件语法高亮，可以方便用户更直观地阅读和修改 Hosts 内容
  • 编辑 hosts 时，可以点击行号可以对行进行快速注释或取消注释

    • SwitchHost镜像下载（已加速）

修改Host

通过SwitchHosts，每次 hosts 有更新都能及时进行更新，免去了手动更新的麻烦
